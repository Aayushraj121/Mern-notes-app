# 🌟 Overview

MERN Secure Notes is a secure and modern full-stack Notes Management application built using the MERN Stack.

Users can:
- ✨ Create Notes
- ✨ Edit Notes
- ✨ Delete Notes
- ✨ Search Notes
- ✨ Manage Personal Notes Securely

This project implements:
- 🔐 JWT Authentication
- 👤 Role-based Authorization
- 🔎 Search & Pagination
- ⚡ Protected REST APIs
- 🎨 Responsive UI

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Aayushraj121/mern-secure-notes.git
```

---

## Install Dependencies

### Backend

```bash
cd server
npm install
```

### Frontend

```bash
cd client
npm install
```

---

## Run Backend

```bash
npm run server
```

---

## Run Frontend

```bash
npm start
```
