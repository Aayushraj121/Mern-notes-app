# 📌 Overview

MERN Secure Notes is a modern full-stack Notes Management application where users can securely create, update, delete, search, and manage notes with JWT authentication and protected APIs.

This project focuses on:
- 🔐 Authentication & Authorization
- 📝 Notes CRUD Operations
- 🔎 Search & Pagination
- 👤 Owner-only Access Control
- ⚡ REST API Integration
- 🎨 Modern Responsive UI

---

# ✨ Features

## 🔐 Authentication
- User Signup & Login
- JWT Authentication
- Password Hashing using bcrypt
- Protected Routes
- Persistent Login Sessions

---

## 📝 Notes Management
- Create Notes
- Edit Notes
- Delete Notes
- User-specific Notes
- Secure Dashboard

---

## 🔎 Search & Pagination
- Search notes by title/content
- MongoDB Regex Search
- Tag Filtering
- Pagination Controls

---

## 👮 Authorization
- Owner-only Edit/Delete
- Admin Moderation
- Protected APIs

# ⚙️ Environment Variables

Create a `.env` file inside the server folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

# 🚀 Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Aayushraj121/mern-secure-notes.git
```

---

## 2️⃣ Install Dependencies

### Backend

```bash
cd server
npm install
```

### Frontend

```bash
cd client
npm install
```

---

## 3️⃣ Run Backend

```bash
npm run server
```

---

## 4️⃣ Run Frontend

```bash
npm start
```
